Renewal: 2 years (platform evolves); renewal = delta exam, not full retake.

## Training content structure (single-source rule)

Playbooks (knowledge/) are the canonical text → Academy lessons are
playbook sections + a practice project + pass criteria. One source, no
drift: updating a playbook updates the course. Practice projects are
ordinary project files with a `goal` block (target KPI + allowed actions)
— graded by the recorder, tamper-proof because results are deterministic.

## Challenge mode
Timed scenarios ("this line loses 12 % OEE — find it in 10 minutes"),
leaderboard by measured result + time. Community content: a challenge is
just a project file — zero authoring cost for us.

## AI Coach training framework (design now, ship post-v1)

The coach teaches THROUGH the product, bound by the same laws:
- **Grounding:** coach explanations must cite CalcValues and recorder facts
  from the live session (the story engine is its knowledge source — it
  narrates measured causality, never invents numbers).
- **Pedagogy ladder:** explain → hint → show → do-with-me; never do-for-me
  during lessons (assessment integrity + Law: augment, never replace).
- **Curriculum linkage:** every coach explanation deep-links the Academy
  lesson covering that concept (knowledge graph edge).
- **Improvement:** coach proposals reuse the WOW-02 measured-suggestion
  service — the AI's advice is simulation-verified before the user sees it.
- **Data:** anonymized, opt-in lesson telemetry tunes hint timing; no
  customer line data ever trains anything without explicit consent.
