# Web App Security 6D Checklist

- [ ] Identity and access reviewed
- [ ] Secrets and stored data reviewed
- [ ] Input, import, HTML, URL, and file handling tested
- [ ] CSP, HTTPS, framing, and network endpoints validated
- [ ] Dependencies and deployment supply chain reviewed
- [ ] Backup, rollback, logging, and incident response documented
- [ ] Findings classified by severity and status
- [ ] Deployment gate decision recorded
