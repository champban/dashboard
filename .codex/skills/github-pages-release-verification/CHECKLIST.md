# GitHub Pages Verification Checklist

- [ ] Publishing branch and folder confirmed
- [ ] Release commit exists on publishing branch
- [ ] `index.html` exists with exact casing
- [ ] Nested routes such as `mobile/index.html` exist
- [ ] GitHub Pages build/deployment status checked
- [ ] Production URLs tested
- [ ] Cache behavior distinguished from deployment defects
- [ ] Critical user flow smoke-tested
- [ ] Rollback commit identified
- [ ] Final release decision recorded
