# Repository Codex Skills

This directory contains reusable, repository-scoped workflows for maintaining and deploying the application safely.

## Available skills

- `github-codex-safe-patch-deploy`
- `webapp-security-6d-audit`
- `github-pages-release-verification`

Always read `PROJECT_CONTEXT.md` before applying a skill. Skills provide the workflow; the project context provides the current architecture, protected behavior, deployment rules, and open backlog.
